IBM Social Learning

An innovative learning solution by IBM

For the past few years I've been working as a Front end and Solr search engineer on IBM Social Learning. ISL is a product that enables users to build up social learning networks, to wrap social learning groups around learning content, and to hold social discussions about the learning content. The solution offers a special status for experts and a highly optimized user experience, especially for video learning content.

Our main clients are Boston's Childrens Hospital, Petrobras, Royal College of Surgeons and Bank of Ireland

I split my time between front end work in Javascript (Dojo), HTML, CSS and some Java based backend development creating RESTful APIs and working on our search functionality using SOLR.
